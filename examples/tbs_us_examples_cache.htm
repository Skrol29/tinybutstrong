Example of plug-in Cache System

this TBS template is W3C compliant

The plug-in Cache System enables you to display a cached page, and to recompute the page with a chosen interval of time. In this example, the time-out is set to 10 seconds. If you recall the page several times, you will see that it is not systematically updated.

Time of merging: [onshow..now;frm='hh:nn:ss']
