Example of sub-template

this TBS template is W3C compliant

Parameter file enables you to include a static sub-template into the main template. Parameter script, with parameter subtpl, enable you to include a dynamic sub-template.

In this example, header and footer are static sub-templates. Headings are also static-sub-templates but they depends to a global variable ($tmpl_article). The login area is a dynamic sub-template. It is managed by a sub-script which can be run under the TBS Subtemplate Mode.

[onload;file='tbs_us_examples_subtpl_header.htm';getbody]
[onload;script=tbs_us_examples_subtpl_login.php;subtpl;getbody]
[onload;file=[var.tmpl_menu];getbody=body] [onload;file=[var.tmpl_article];getbody]
[onload;file='tbs_us_examples_subtpl_footer.htm';getbody]
