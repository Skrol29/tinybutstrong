Example of parameter serial

this TBS template is W3C compliant

Parameter serial enables you to display several records in a main block. For that, you must use sub-blocks.

Example of parameter serial used for a multi-columns display:

Position / Title
> [bx_1.rank]
[bx_1.title;block=td]
 [bx_2.rank]
[bx_2.title;block=td;.]
 [bx_3.rank]
[bx_3.title;block=td;.]
[bx;block=tr;serial]
  This line is optional, it enables you to
define the empty cell.		 [bx_0;block=td]
[bx;block=tr;serial]

Example of parameter serial used for a complex display:

Position / Title
>
[bz_1.rank] [bz_1.title;block=td]
[bz_3.rank] [bz_3.title;block=td;.]
 
[bz_2.rank] [bz_2.title;block=td;.]
 
[bz;block=tr;serial]
