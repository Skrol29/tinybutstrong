Example of parameter att

this TBS template is W3C compliant

Parameter att allows to remotly modify the attributs of any XML/Html tags.
This can be very precious when it is not easy to reach the source code of the template (OpenOffice, Ms Office 2007, Dreamweaver, ...). Or simply if you want to code an Html template which is W3C compatible.

Without att Using att Comment
