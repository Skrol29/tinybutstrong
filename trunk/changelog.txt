
TinyButStrong 1.8.0 beta

2011-10-25, change-log:
=======================

- bug with PHP 4 : PDO::FETCH_ASSOC is not recognized
- bug: A string [] appears in the contents when parameter "att" is used with a "parentgrp" block.
- new feature ExtendedMethods: enables plug-in coders to add new TBS methods
- new feature VarRef + SetVarRef(): enables to change [var] field's reference.
- new feature IncludePath: similar to PHP include_path for template, scripts and sub-templates.
- new method SetOption() / GetOption() : a shorthand for many TBS options
- new method SetTplFrms(): enables you to add format alaises at the PHP side
- new native database support: Zend Adapater
- new value for parameter "ope": upper, lower, upper1, upperw, utf8
